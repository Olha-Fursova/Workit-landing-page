### Опис завдання

Ваше завдання — створити цю цільову сторінку та зробити її максимально схожою на дизайн.

Ви можете використовувати будь-які інструменти, які вам подобаються, щоб допомогти вам виконати завдання. Тож, якщо у вас є щось, що ви хотіли б потренуватися, сміливо спробуйте.

Ваші користувачі повинні мати можливість:

- Переглядати оптимальний макет залежно від розміру екрана свого пристрою
- Переглядати стани наведення курсора для інтерактивних елементів

### Використані технології

- HTML5 для розмітки
- CSS3 стилізація
- Flexbox
- Grid

## Висновки

Дане завдання було досить цікавим та в міру легким. Розташування об'єктів відповідно до пропорцій на макеті, невелика робота з псевдоелементами та автопідрахунком, зміна layout при різному розширенні екрану - основні виклики, але досить прості для роботи з нимию. Основною задачею, як завжди, поставила перед собою написання чистого, зручного та масштабованого кодую

У роботі дотримувалась розділення стилів на різні файли для кожної секції. Наступного разу б залучила використання Vite та можливість розділення html коду так само по секціях з посиланням на них в основному файлі для кращої структуризації коду.

### Мої досягнення

Я вже набагато швидше знаходжу помилки та баги і швидко знаходжу їм рішення. Побудова комьінованого лейауту з чергуванням flexbox та grid в залежності від розширення екрану та розташування елементів на макеті є легкою та природньою задачею, що не викликає сум'яття.
